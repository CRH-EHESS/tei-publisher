TEI ODD extensions for processing models

An implementation of the TEI Simple Processing Model in XQuery. Use this app to experiment and try out various ODDs containing processing model instructions. The documents shown below are test documents only. You can upload additional documents/ODDs by using eXide.

Once your happy with an ODD, use the "App Generator" to create a standalone app based on it. Please check the Early English Books Online (EEBO) app to see an example of a real application which was generated this way.
